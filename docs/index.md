# Cognitive Model Protocol (CMP)

CMP is an experimental open protocol for representing explicit, evidence-based, uncertainty-aware working models of human-AI collaboration.

Current status: **0.1.0-draft**, seeking criticism.

CMP models collaboration with a user, not the user's identity. It is not a personality test, diagnosis, advertising profile, or finished standard.
