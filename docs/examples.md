# Examples

The repository includes two draft YAML examples.

The canonical protocol definition is [reference/current.md](https://github.com/filinils/cognitive-model-protocol/blob/main/reference/current.md).

## Minimal Example

[examples/minimal-example.yaml](https://github.com/filinils/cognitive-model-protocol/blob/main/examples/minimal-example.yaml)

A small generic example with two hypotheses. It shows the required root fields, a `collaboration_model`, evidence records, confidence values, uncertainty, falsifiers, signals, revision history, and user controls.

Use this as the simplest starting point for understanding the shape of a CMP document.

## Systems-Synthesist Example

[examples/systems-synthesist-example.yaml](https://github.com/filinils/cognitive-model-protocol/blob/main/examples/systems-synthesist-example.yaml)

An illustrative fictional and anonymized example of a person who tends to reason across domains, seek synthesis, prefer model-building over fixed conclusions, value feedback loops, and respond well to constructive resistance.

This example is not a template for everyone. It is included to show how richer collaboration hypotheses and interaction patterns might be represented while preserving uncertainty and falsifiers.
