# Governance

CMP is currently a draft protocol maintained by Filip Nilsson.

The canonical protocol definition is [reference/current.md](reference/current.md).

The initial governance model is intentionally simple:

- Maintainer: Filip Nilsson.
- Status: Draft protocol.
- Changes are proposed through GitHub issues and pull requests.
- Larger changes should be proposed as CMP RFC documents later.
- The protocol should not be treated as stable until a future version explicitly says so.

Governance should evolve if the project gains contributors, implementations, or broader adoption.
