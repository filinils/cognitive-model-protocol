# Changelog

All notable changes to CMP will be documented in this file.

## 0.2.0-draft - 2026-06-30

- Added normative CMP Conformance Floor requirements.
- Added L1-L3 conformance levels.
- Added conformance disclosure format, JSON Schema, and worked example.
- Added extension framework and `ext:matching` v0.1 requirements.
- Clarified that matching requirements apply to implementations and that CMP defines no matching algorithm.

## 0.1.0-draft - 2026-06-28

- Added initial README, rationale, and draft specification.
- Added draft JSON Schema for CMP documents.
- Added minimal and systems-synthesist illustrative YAML examples.
- Added contribution, governance, conduct, citation, notice, and licensing files.
